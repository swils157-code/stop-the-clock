# Stop the Clock 🎯

A dead-simple reaction game: tap the green button to start a counting stopwatch, then tap **STOP** as close to the target time as you can. Get within 0.05s to win. Each round gives a new target.

Also known online as the **Stopwatch Challenge** / **10-Second Challenge** — the viral "stop the timer on exactly 10.00" game.

## Run locally
Just open `index.html` in any browser. No build step, no dependencies.

## Deploy
Static site — deploys to Vercel as-is (no config needed).

## Features
- Visible counting timer with click sound on every tap (Web Audio)
- Randomized round targets, accuracy feedback, best score + win streak
- Mobile-first, works with tap, click, Space, or Enter
